Metro Style Social Network Widget

== Description ==

A light weight wordpress widget that displays Metro Style social network icons in Sidebar and links to your social network profiles.

Add button option will display the Like / Follow button on the Social icons so that your users can immediately Like / Follow your page or profile without leaving your site.

Since every one use Facebook and Twitter, we made them as default and other network as optional.

= Update =

Color Picker added, Now you can set any color to any icons in the widget.

On Users request Google+ and RSS Feed are now optional.

If needed you can display Google+, Linkedin, Youtube, RSS Feed and Pinterest else choose don't show option in widget settings.

Responsive design, Plugin will automatically fit within your siderbar width else you can specify width for the widget.

== Installation ==

1. Download the zip file and upload it using upload option in your wordpress dashbord under Plugins-> Add New.

2. Activate the plugin.

3. Goto Appearence-> widget, Now just drag and drop the Metro Style Social widget in sidebar.

4. Type the ID / Username of your Social Networks sites and save it.

5. That's it, Hope you enjoy the Plugin.
